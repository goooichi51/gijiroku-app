# UI/UXワイヤーフレーム仕様書

## デザイン方針

- **ナビゲーション**: ボトムタブバー（3タブ）— iOS標準パターン、片手操作に最適
- **録音開始**: 中央のフローティング録音ボタン — ワンタップで開始
- **カラー**: ライト/ダークモード両対応（iOS標準に準拠）
- **フォント**: SF Pro（iOS標準）

### 2. 録音画面

```
┌─────────────────────────────┐
│ ✕ 録音中             01:23:45│  ← 経過時間表示
├─────────────────────────────┤
│                             │
│  ～～～～⌇⌇⌇～～～～～⌇⌇  │  ← 波形表示（リアルタイム）
│                             │
├─────────────────────────────┤
│                             │
│ 「それでは次の議題に移り   │  ← リアルタイム文字起こし
│  ましょう。来月のキャンペ  │     プレビュー（P1機能）
│  ーンについてですが...」   │
│                             │
│ 話者A: 「売上目標は前年比  │
│  120%を目指したいと思い    │
│  ます」                    │
│                             │
│                       ↓自動 │  ← 自動スクロール
├─────────────────────────────┤
│                             │
│     ⏸          ⏹           │  ← 一時停止 / 停止ボタン
│    一時停止     停止         │
│                             │
└─────────────────────────────┘
```

**仕様**:
- ✕ボタン → 「録音を破棄しますか？」確認ダイアログ
- 一時停止 → 再開ボタンに切り替わる
- 停止 → 議事録作成画面に遷移
- バックグラウンドに移行しても録音継続（画面上部にオレンジインジケーター）
- 波形はリアルタイムで描画
- **最大録音時間: 4時間**。4時間経過で自動停止し議事録作成画面に遷移
- 残り10分で「残り10分で自動停止します」のバナー通知を表示

## Apple Watch画面仕様

### Watch: メイン画面

```
    ┌───────────┐
    │           │
    │   ● REC   │     ← 大きな録音ボタン
    │           │
    │  タップで  │
    │  録音開始  │
    └───────────┘
```

### Watch: 録音中

```
    ┌───────────┐
    │  ● 録音中  │
    │           │
    │  01:23:45 │     ← 経過時間
    │  ～～～～  │     ← 簡易波形
    │           │
    │   ⏹ 停止  │     ← 停止ボタン
    └───────────┘
```

**Watch録音の自動管理**:
- **最大録音時間: 4時間**。4時間で自動停止
- **30分ごとに継続確認を表示**:

```
    ┌───────────┐
    │  30分経過  │
    │           │
    │ 録音を    │
    │ 続けますか│
    │           │
    │ [続ける]  │     ← タップで継続
    │ [停止]    │
    └───────────┘
```

- **5分間応答なし → 自動停止**（バッテリー浪費防止）
- 停止後はiPhoneへの自動転送に遷移

    │ 転送中... │
    │           │

## ユーザーフロー

```
[初回起動]
  │
  ▼
[オンボーディング（3画面）]
  │
  ▼
[Apple IDログイン / スキップ]
  │
  ▼
[ホーム画面]
  │
  ├── 録音ボタンタップ
  │     ▼
  │   [録音画面] ← リアルタイム文字起こし表示
  │     │
  │     ▼ 停止
  │   [議事録作成画面]
  │     │ テンプレート選択 + メタ情報入力
  │     │ 文字起こし処理（バックグラウンド）
  │     ▼
  │   [AI議事録生成] → Gemini 3 Flash API
  │     │
  │     ▼
  │   [議事録詳細画面]
  │     ├── 編集
  │     ├── PDFプレビュー
  │     └── 共有（LINE等）
  │
  ├── 検索タブ → 全文検索
  │
  └── 設定 → アカウント / プラン / データ管理
```

## 競合UIからの学び

| 学び | 参考元 | 採用方針 |
|------|--------|---------|
| 録音ボタンは中央に大きく配置 | Just Press Record, Notta | 採用: タブバー中央にフローティング |
| 要約/全文の2タブ構成 | Otter.ai | 採用: セグメントコントロール |
| PiP文字起こし表示 | Texter | v2.0で検討 |
| ダブルタップで録音開始（Watch） | Genspark, Just Press Record | 採用: watchOS 11対応 |
| 20種のAIテンプレート | Notta, Texter | MVP: 4種、v2.0でカスタム対応 |
| バッチエクスポート | Notta | v2.0で検討 |
